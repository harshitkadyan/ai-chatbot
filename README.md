# AI Chatbot Project

## Overview
This is a simple AI Chatbot project that allows users to interact with an AI-powered assistant through a web interface. The chatbot backend is built using **Python** and **Flask**, and the frontend uses **HTML**. The project is fully deployed on a live server using **Render**.

## Project Management
The development of this project was managed using **Treelo** to ensure proper task tracking, organization, and smooth collaboration. All features and updates were planned and tracked systematically.

## Features
- Interactive chat interface
- Backend powered by Python and Flask
- Frontend implemented in HTML
- Fully deployed on a live server
- Proper project management using Treelo

## Tech Stack
- **Backend:** Python, Flask  
- **Frontend:** HTML  
- **Project Management:** Treelo  
- **Deployment:** Render

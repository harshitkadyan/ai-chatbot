# AI Chatbot

## Overview
This is a simple AI chatbot built using Python and Flask. It takes user input and returns a response via a web interface.

## Features
- Chat interface with input box and send button
- Backend API using Flask
- Deployed on Render

## Tech Stack
- Python
- Flask
- HTML, CSS, JavaScript
